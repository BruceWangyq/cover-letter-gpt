### ✨ Key Features

- Next.js 13 and Tailwind CSS
- GPT-3 API
- Fully Responsive
- Dark Mode
- SEO Optimized
- Vercel Deployment

## Running Locally

1. Install dependencies using yarn:

```sh
yarn install
```

2. Copy `.env.example` to `.env.local` and update the variables.

```sh
cp .env.example .env.local
```

3. Start the development server:

```sh
yarn run dev
```
